# Contributing

## Setup
1. Install Node dependencies with `npm install`
2. Run tests with `npm test`
3. Build with `npm run build`

## Expectations
- Keep the app local-only by default
- Do not commit real Floatplane session data, browser profiles, HAR files, or probe payloads
- Prefer small, reviewable pull requests
- Add or update tests when behavior changes

## Areas of focus
- Linux desktop runtime behavior
- Floatplane auth/session resilience
- Playback relay stability
- Background auto-watch scheduling
